# SDSS Celestial Object Classification

A machine learning project that classifies celestial objects into Stars, Galaxies, and Quasars using Sloan Digital Sky Survey (SDSS) data.

## Features

* Data preprocessing and feature scaling
* Label encoding of target classes
* Decision Tree Classification
* K-Nearest Neighbors (KNN)
* Model evaluation using classification reports
* Confusion matrix visualization
* Decision tree visualization

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

## Dataset

Sloan Digital Sky Survey (SDSS) dataset containing astronomical observations and object classifications.

## Results

The trained models achieved high classification accuracy in distinguishing Stars, Galaxies, and Quasars.
